# Excel-Financial-Dashboard

Excel Financial Analysis Dashboard
An interactive Excel dashboard analyzing departmental budget vs. actual expenses across 12 months to identify spending patterns and support cost control decisions.
Tools Used

Microsoft Excel (Advanced)
Power Query
Pivot Tables & Slicers
Charts & Conditional Formatting

What I Built

Cleaned and transformed raw financial data using Power Query
Built Pivot Tables to compare budgeted vs. actual spend by department and month
Created an interactive dashboard with slicers for dynamic filtering by department and time period
Visualized profit trends and spending variances using bar and line charts

Key Insight
Identified that 3 departments consistently exceeded their budget by 15%+, enabling targeted cost control recommendations.
